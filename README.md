# 🛒 Product Explorer

A modern E-commerce product browsing experience built with Next.js 15, React, and TypeScript.
Includes search-as-you-type, login experience, and polished UI components.

## Developing

### Install
```
yarn
```

### Build
```
yarn build
```

### Running Locally
```
yarn dev
```
